# RoyalStay Hotel Management System

React + Vite hotel management system with online booking functionality.

## Features
- Responsive hotel landing page
- Room listing with search and filter
- Online booking form with validation
- Auto billing calculation based on check-in and check-out dates
- Booking save using localStorage
- Edit and delete booking
- Admin dashboard cards
- Booking table and guest management view
- Contact section

## Run Project
```bash
npm install
npm run dev
```

Open the local Vite URL shown in terminal.
